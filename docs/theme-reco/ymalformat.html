    YAML front matter

    Elio 8/19/2021

    # 常用的变量

    # title (opens new window)

    • description: 文章标题，放弃通过一级目录定义标题的方式，改在 Front Matter 中定义。

    # date (opens new window)

    • description: 文章创建日期，格式 2019-08-08 或 2019-08-08 08:08:08。

    # sidebar (opens new window)

    • description: 是否开启侧边栏。

    # categories (opens new window)

    • description: 所属分类。

    # tags (opens new window)

    • description: 所属标签。

    # keys (opens new window)

    • description: 文章加密密码。

    # publish (opens new window)

    • description: 文章是否发布。

    # sticky 1.1.2+ (opens new window)

    • description: 文章置顶。
    • type: number
    • sort type: 降序，可以按照 1, 2, 3, ... 来降低置顶文章的排列优先级

    # 其他的变量 (opens new window)

    另外还有一些Vuepress 默认主题的变量例如prev, next，请移步 官方文档 (opens new window)
